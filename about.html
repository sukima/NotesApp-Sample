About NotesApp

Notes App

By Devin Weaver

I came across a tutorial by Jorge Ramon which offered a small sample application to store notes.

I took this and designed this application. It illustrates using jQuery, jQuery Mobile, jQuery-mobile router, Handlebars.js, jStorage, Google Bookmark Bubble, and CoffeeScript. I use hem as the package manager and Jasmine for unit testing. I also created a very simple data Model, View, and Controller. This was an exercise in doing things by hand. I recommend using an established MV* framework such as Backbone or Ember.

Source Code available on GitHub. Feel free to clone/fork and try running the tests.
